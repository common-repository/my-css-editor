== Description ==

This adds the Style Sheet definition that I set while it is <head>-</head> to use Style Sheet besides the theme being used now when contributing. 

== Installation ==

1. Upload `my-css-editor.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. If the Style Sheet definition that you want to use is set and the update button is pressed, it is possible to use it

== Frequently Asked Questions ==

= Where is the Style Sheet definition preserved? =

It is preserved in the data base by the name described in the left of the text area on the definition screen. 
It is a name of "mse_option.css.1" now. 

= Is it good if it does very to delete the definition? =

"delete" Please press the button. It is deleted from the data base.
